## 细分着色器（Tessellation Shader）

> ⚠️ **WebGL（1 和 2）都没有细分着色器**（和几何着色器一样，从未进入 WebGL 规范）。本 Demo 用 **CPU 递归细分**来模拟它的核心思想。

这个 Demo 从一个**棱角分明的二十面体**（20 个三角形的粗糙"控制网格"）出发，随着「细分级别」滑块升高，每个三角形被切成更多小三角形，并投影到球面上——多面体逐渐变成一个**光滑的球**。打开线框能清楚看到三角形越分越细。

### 一、细分着色器在管线里的位置

```
顶点着色器 → 细分控制(TCS) → [固定细分器] → 细分求值(TES) → [几何着色器] → 光栅化 → 片元着色器
```

细分阶段其实是**三个部分**协作：

1. **TCS（Tessellation Control Shader，控制着色器）**：逐面片运行，决定这个面片要被细分成多细——输出**细分级别（tessellation level）**，分 inner（内部）和 outer（边缘）两种。
2. **固定功能细分器（Tessellator）**：不可编程，按 TCS 给的级别，机械地把面片切成大量小三角形（只生成**重心坐标**，还没有真实位置）。
3. **TES（Tessellation Evaluation Shader，求值着色器）**：逐**新生成的顶点**运行，根据重心坐标算出它的**最终位置**——这一步决定了细分出的面到底贴在什么曲面上。

本 Demo 的对应关系：

| 真实管线 | 本 Demo（CPU 模拟） |
|---|---|
| TCS 输出细分级别 | 「细分级别」滑块 |
| 固定细分器切三角形 | 每级把 1 个三角形切成 4 个（取三边中点） |
| TES 求每个新顶点位置 | 把新顶点**归一化到球面** `P → normalize(P)·r` |

### 二、细分级别 → 三角形数

二十面体有 20 个面，每细分一级三角形数 ×4：

```
级别 0：20      （原始控制网格，能明显看出是个多面体）
级别 1：80
级别 2：320
级别 3：1280
级别 4：5120    （已非常接近光滑球）
```

面板上的「三角形数」会随级别实时变化。**关键点**：你传给 GPU 的永远只是那 20 个三角形的控制网格，细分是**在 GPU 上动态发生**的——这就是细分着色器最大的价值：用极少的数据描述一个面，由 GPU 按需生成任意精细的几何。

### 三、"细分"和"求值"是两件事

取消勾选「投影到球面」：此时只做**细分**（切三角形），但**不做求值**（新顶点保持在原三角形平面上的线性中点）。你会看到——三角形数一样在涨，但形状**始终是那个多面体**，永远变不圆。

这说明：

- **细分器**只负责"把面切碎"，切出来的点默认还在原面上；
- **真正把模型变光滑的是 TES**——它把每个新顶点放到目标曲面（这里是球，真实项目里常是**贝塞尔/B 样条曲面**或**配合高度图做置换**）。

勾上投影 = TES 把每个新顶点推到球面 → 切得越碎、越逼近真球。

### 四、和 LOD、几何着色器的区别

- **vs LOD（层次细节）**：LOD 是"准备好几套不同精度的模型，按距离选一套"；细分是"只给一个粗模型，GPU 按需现场生成精细网格"。细分级别甚至可以**逐面片连续变化**（近处面片高、远处面片低），常用来做**动态 LOD**。
- **vs 几何着色器**：几何着色器逐图元、能增删任意图元但数量有限、不擅长大规模均匀细分；细分着色器是**专门为"大规模规则细分曲面"设计**的固定+可编程混合阶段，吞吐高得多。

### 五、WebGL 没有它，怎么办

- 像本 Demo 一样 **CPU 预先细分**好再上传（适合静态模型）；
- 顶点着色器里做有限的顶点位移近似（如地形置换）；
- 真正的 GPU 动态细分要靠**桌面 OpenGL 4 / Direct3D 11+** 的细分阶段，或 **WebGPU** 的 compute 管线。

### 小结

- 细分着色器把一个**粗控制网格**在 GPU 上**动态细分**成精细曲面，三大件：TCS 定级别、细分器切面、TES 求最终位置；
- 三角形数随级别按 4ⁿ 暴涨，但上传的数据始终是那个小控制网格；
- **细分 ≠ 求值**：切碎只是第一步，TES 把新顶点放到目标曲面才让模型变光滑（本 Demo=球面，取消投影则仍是多面体）；
- 与 LOD（选模型）、几何着色器（增删图元）定位不同；
- **WebGL 无细分着色器**，本 Demo 用 CPU 递归细分模拟，真正 GPU 细分要靠桌面 API / WebGPU。
