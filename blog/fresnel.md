## 菲涅尔效果（Fresnel Reflection）

这个 Demo 用一个大的球体来孤立演示**菲涅尔反射**——即物体表面的反射率**随观察角度变化**的现象。球体中央颜色暗（正对相机），边缘有一圈光晕（斜看/掠射角），这就是菲涅尔效应的直观表现。

### 一、什么是菲涅尔效应

站在湖边看水：

- **低头看脚下**（视线垂直于水面，N·V ≈ 1）：水面几乎透明，能看到水底的石头；
- **抬头看远处**（视线几乎平行于水面，N·V ≈ 0）：水面变成一面镜子，倒映天空和远山。

同一个水面，反射率随观察角度剧烈变化——这就是**菲涅尔效应**。

在所有光滑表面（玻璃、水、塑料、金属、皮肤的油脂……）都成立：**掠射角（grazing angle）反射最强，法线方向反射最弱**。

### 二、Schlick 近似公式

完整的菲涅尔方程涉及光的偏振和复数折射率，计算复杂。实时渲染中几乎都使用 **Schlick 近似**（由 Christophe Schlick 在 1994 年提出）：

```
fresnel = F0 + (1 - F0) × (1 - N·V)^power
```

其中：

| 符号 | 含义 |
|------|------|
| **N** | 表面法线（归一化） |
| **V** | 视线方向（从片元指向相机，归一化） |
| **N·V** | 法线与视线的点积。N·V = 1 表示正对着看，N·V = 0 表示几乎平行 |
| **F0** | 基础反射率（base reflectance）——正对表面（N·V=1）时的反射率 |
| **power** | 指数，控制从"低反射"到"高反射"的过渡速度。Schlick 经典值 = 5 |

向量约定：`N = normalize(vWorldNormal)`, `V = normalize(uViewPos - vWorldPos)` —— V 从片元指向相机。

> 本 Demo 的着色器使用了 `abs(dot(N, V))`，这样球体背面（法线背对相机）也会出现边缘光，形成**环绕整个球体的光环**，视觉效果更强。

### 三、F0 —— 材质的基础反射率

F0 回答一个问题：**正对着看（N·V=1）时，反射多少光？**

```
F0 ≈ 0.02        水
F0 ≈ 0.04        玻璃、塑料、皮肤（典型电介质 dielectric）
F0 ≈ 0.08        宝石
F0 ≈ 0.17        钻石
F0 ≈ 0.5 ~ 0.7   钢、铝
F0 ≈ 0.8 ~ 1.0   金、铜（典型金属 conductor）
```

**电介质**（绝缘体）F0 很低（0.02~0.08），无论什么颜色都用同一个灰度的 F0。
**金属**（导体）F0 很高（0.5+），而且**有色**——黄金反射的是金色的光，所以 F0 是 `vec3(1.0, 0.71, 0.29)` 而非单一灰度。

> **试试看**：拖动「F0」滑块从 0.0 扫到 1.0。F0=0.04（玻璃/塑料）时中心很暗、只有边缘亮；F0=0.8（金属）时整个球都亮起来——金属几乎在全角度都反光。

### 四、Power —— 边缘光有多宽

`(1 - N·V)^power` 决定了"反射率从 F0 上升到 1.0"的过渡速度：

```
power = 1   →   pow(x, 1) = x     →  线性过渡，光晕很宽
power = 5   →   pow(x, 5)           →  经典 Schlick，光晕窄而锐（推荐）
power = 8   →   pow(x, 8)           →  光晕极其锐利，几乎只在最边缘
```

power 越大，高反射区域越窄、越挤向边缘。Schlick 选择 5 是因为它在视觉上最接近完整菲涅尔方程的曲线。

> **试试看**：把 Power 拖到 1.0 看看宽光晕，再拖到 5.0 看看边缘有多锐，最后拖到 8.0——你会发现"边缘光"变成了极其细的一圈环。

### 五、着色器核心代码

本 Demo 的片元着色器逻辑（简化）：

```glsl
vec3 N = normalize(vWorldNormal);
vec3 V = normalize(uViewPos - vWorldPos);
float NoV = abs(dot(N, V));

// Schlick Fresnel
float fresnel = uF0 + (1.0 - uF0) * pow(1.0 - NoV, uPower);

// 颜色 = 在基底色和边缘光色之间按 fresnel 系数混合
gl_FragColor = mix(uBaseColor, uRimColor, fresnel * uEnable);
```

`uEnable` 是 1.0 或 0.0，对应「启用菲涅尔」复选框。关掉后 `fresnel*uEnable = 0`，球体退化为纯基底色，对比出菲涅尔的贡献。

### 六、为什么 PBR 要用菲涅尔

在 PBR（基于物理的渲染）中，菲涅尔是 Cook-Torrance BRDF 的三大核心项之一（另外两项是 NDF 法线分布和 Geometry 几何遮蔽）。菲涅尔决定：
- **镜面反射的强度**随观察角度变化——掠射角时镜面反射最强（所有光滑表面在边缘都像镜子）
- **漫反射的能量补偿**——掠射角时镜面反射拿走了大部分能量，漫反射要相应减弱

这就是为什么 PBR 的金属材质在边缘看起来特别亮——菲涅尔把镜面反射推到了接近 100%。

### 小结

- 菲涅尔 = 光滑表面的反射率**随观察角度变化**：正看弱，斜看强；
- Schlick 近似：`F0 + (1-F0) × (1-N·V)^5`，是实时渲染的标准实现；
- F0 区分材质类型：电介质 ~0.04，金属 0.5~1.0；
- Power=5 是经典值，改变它可看到边缘光晕宽度的变化；
- 拖动旋转球体（鼠标左键拖主窗口），可以从不同角度直观感受菲涅尔系数的变化。
